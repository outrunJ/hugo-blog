# 约定
## 巴科斯范式
    "word"  表示字符本身
    double_quote    表示"本身
    ""      外表示语法字符
    <>      表示必选项
    []      表示可选项
    {}      表示可重复0到多次
    |       表示or
    ::=     表示“被定义为”
