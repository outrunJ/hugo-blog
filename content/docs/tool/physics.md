# HHKB HyBrid
	生产日期 20210108
	购买日期 202104
## 配对方式
	1.按Fn + q键进入配对模式。
	2.再按Fn + z + ~ 键清空键盘配对信息,键盘自动关闭。
	3.长按开关键重新打开键盘后按Fn + q键进入配对模式,再按Fn + Ctrl + 1/2/3/4指定一个序号。
	4.然后关闭电脑蓝牙功能再打开进行搜索。
## 切换
	Fn + Ctrl + 1/2/3/4
	Fn + Ctrl + 0
