# 标题
    # 到 ######开头    # 分别表示一号 到 六号 标题

# 字体
    **a**或__a__   # 加粗
    *a*或_a_ # 斜体
    ***a***或___a___ # 加粗斜体
    ~~a~~   # 删除线


# 引用
    >a
    >>b # 不断增加>来多层引用

# 分割线
    --- # 三个及以上
    *** # 三个及以上，与---显示无分别

# 图片
    ![alt内容](url "title内容") # alt显示在图片下，title在鼠标悬停时显示

# 超链接
    [文本](url)
    <a href="url" target="_blank">文本</a>    # 可以用a标签指定target，在新页面显示

# 列表
    -或+或*开头 # 无序列表- 
    数字加点开头  # 有序列表，行数自动
    多空格（至少两个）加列表开头  # 嵌套列表

# 表格
    标题1|标题2|标题3 # 默认居中对齐
    -|-|-   # -可以多个，:-控制标题和内容左对齐，:-:居中，-:右对齐
    1|2|3   ＃ 默认左对齐

# 代码
    `a + b;`    # 单选代码
    ```
    function a(){
    }
    ``` # 多行代码
